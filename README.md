
## 文件结构

```shell
.
├── build  项目构建配置
├── public  打包所需静态资源
└── src
    ├── api  AJAX请求
    └── assets  项目静态资源
        └── iconfont  自定义图标资源
    ├── components  业务组件
        └── left-menu  菜单框架
    ├── config  项目运行配置
    ├── css  样式配置
    ├── pages  页面文件
    ├── router  路由配置
    ├── store  Vuex配置
    └── utils  封装工具函数
```
